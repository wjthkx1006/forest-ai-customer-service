# 森林二手手机智能客服

一个基于AI的二手手机智能客服系统，提供快速、准确的问答服务。

> **只需修改DASHSCOPE_API_KEY即可使用！**

后续可部署到云服务器（如streamlit cloud,腾讯云或阿里云等），实现24小时不间断服务。

### 第1步：获取API密钥

1. 访问[阿里云DashScope控制台](https://dashscope.console.aliyun.com/)
2. 注册/登录阿里云账号
3. 创建API密钥（选择Qwen系列模型）
4. 复制您的`DASHSCOPE_API_KEY`

### 第2步：配置项目

1. 打开项目中的`.env.example`文件
2. **只需修改这一行**：
    DASHSCOPE_API_KEY=您的API密钥
   再将.env.example文件重命名为.env
3. 其他配置项已有默认值，无需修改

### 第3步：运行应用

```bash
# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run main.py
```

 **完成！** 浏览器会自动打开智能客服界面，您可以立即开始使用。

##  配置说明

### 环境变量配置（.env文件）

项目根目录下的`.env`文件包含所有配置：

```bash
# 阿里云DashScope API密钥（必须修改）
DASHSCOPE_API_KEY=

# 向量数据库存储路径（默认即可）
CHROMA_DB_PATH=./chroma_db

# 客服联系信息（可自定义）
WECHAT_ID=123456789
PHONE_NUMBER=123456789
```

**配置说明：**

1. **DASHSCOPE_API_KEY**（必须修改）
   - 获取地址：https://dashscope.console.aliyun.com/
   - 新用户有免费额度
   - 支持Qwen系列模型

2. **CHROMA_DB_PATH**（默认即可）
   - 向量数据库存储路径
   - 首次运行会自动创建
   - 如需重置，删除此目录后重启应用

3. **WECHAT_ID / PHONE_NUMBER**（可选）
   - 客服联系信息
   - 在回答中可能会引用
   - 可设置为您的实际联系方式

### 模型配置

配置位于`config.py`文件，无需修改：

```python
# AI模型配置
AI_MODEL = "qwen-turbo"           # 使用的AI模型
EMBEDDING_MODEL = "text-embedding-v2"  # 嵌入模型
TEMPERATURE = 0.3                 # 创造性（0-1，值越低越保守）
MAX_TOKENS = 800                  # 最大生成token数

# 向量数据库配置
CHROMA_COLLECTION_NAME = "faq_knowledge"  # 集合名称
SIMILARITY_THRESHOLD = 0.7        # 相似度阈值（0-1）

# 缓存配置
CACHE_TTL = 3600                  # 缓存有效期（秒）
MAX_CACHE_SIZE = 100              # 最大缓存条目数
```

**可调整参数：**

- `TEMPERATURE`：控制回答的创造性（0.1-0.5推荐）
- `MAX_TOKENS`：控制回答长度（500-1000推荐）
- `SIMILARITY_THRESHOLD`：知识库匹配阈值（0.6-0.8推荐）

### 知识库配置（faq.json）

`faq.json`文件包含预定义的问答对：

{
  "questions": [

**自定义知识库：**

2. 重新构建向量数据库：`python build_vector_db.py`
3. 重启应用

##  常见问题解答

### Q1: 我只需要修改DASHSCOPE_API_KEY吗？

**A:** 是的！项目已完全配置好，您只需：

1. 获取阿里云DashScope API密钥
2. 修改`.env`文件中的`DASHSCOPE_API_KEY`
3. 运行`streamlit run main.py`

其他所有配置都有合理的默认值。


### Q2 运行时报错"ModuleNotFoundError"怎么办？

**A:** 确保已安装所有依赖：

```bash
pip install -r requirements.txt
```

如果仍有问题，尝试：

```bash
# 更新pip
pip install --upgrade pip

# 重新安装依赖
pip install -r requirements.txt --force-reinstall
```

### Q3 应用启动后无法访问怎么办？

**A:** 检查：

1. 端口是否被占用（默认8501）
   ```bash
   # 使用其他端口
   streamlit run main.py --server.port 8502
   ```
2. 防火墙是否允许访问
3. 浏览器是否支持（推荐Chrome/Firefox）

### Q4 AI回答速度慢怎么办？

**A:** 系统已优化响应速度，如果仍感觉慢：

1. 首次回答会稍慢（需要初始化）
2. 后续回答会使用缓存，速度更快
3. 检查网络连接
4. 查看`logs/`目录下的日志文件

### Q5 如何自定义知识库？

**A:**

1. 修改`faq.json`文件，添加您的问答对
2. 重新构建向量数据库：
   ```bash
   python build_vector_db.py
   ```
3. 重启应用

### Q7: 如何修改客服信息？

**A:** 修改`.env`文件：

```bash
WECHAT_ID=您的微信ID
PHONE_NUMBER=您的联系电话
```
